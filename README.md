# 🐾 Subramani Pets Shop – E-Commerce Website

A modern, responsive E-commerce Pet Shop website built using **HTML, CSS, and JavaScript**.  
This project includes a dynamic product system, working Add to Cart functionality, and localStorage integration.

---

## 🚀 Features

- 🛍️ Product listing for Dogs, Cats, and Fish
- 🛒 Real Add to Cart system
- 🔢 Cart count displayed in navbar
- 💾 Cart data saved using localStorage
- 📱 Fully mobile responsive design
- 🎨 Clean modern UI with consistent theme
- 📂 Organized folder structure (assets categorized)

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Responsive Layout, Grid)
- JavaScript (DOM Manipulation, localStorage)
- Git & GitHub (Version Control)
